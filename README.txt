== Description ==

A plugin to extend WordPress functionalities to enable people create voluntary clusters and help each other in covid19 pandemic.

see https://covid19help.eu


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `pooling.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('pooling_hook'); ?>` in your templates
